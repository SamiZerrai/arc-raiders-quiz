# Images des Robots ARC

## Instructions

Pour que le quiz affiche les images des robots, vous devez ajouter les fichiers image dans ce dossier.

### Format des fichiers

Chaque fichier image doit être nommé exactement comme le nom du robot (sensible à la casse) avec l'extension `.webp`.

**Format requis**: WebP

### Liste des fichiers nécessaires:

- `Tick.webp`
- `Pop.webp`
- `Fireball.webp`
- `Surveyor.webp`
- `Turret.webp`
- `Sentinel.webp`
- `Snitch.webp`
- `Wasp.webp`
- `Hornet.webp`
- `Rocketeer.webp`
- `Leaper.webp`
- `Bastion.webp`
- `Spotter.webp`
- `Bombardier.webp`
- `The Queen.webp`
- `Harvester.webp`
- `ARC Probe.webp`

### Exemple de structure:

```
public/
  images/
    Tick.webp
    Pop.webp
    Fireball.webp
    ...
```

### Quand les images s'affichent-elles?

Les images des robots s'affichent:
- ✅ Quand vous trouvez la bonne réponse
- ❌ Après avoir épuisé les 3 tentatives (pour montrer quel était le bon robot)

### Note

Si une image n'est pas disponible, un placeholder sera affiché avec le nom du robot.
